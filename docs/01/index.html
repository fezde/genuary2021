Prompt

// TRIPLE NESTED LOOP

Interpretation

The composition is built up from three levels of loops:
  1. The written word "loop"
  2. An animation that rotates the image of the first level
  3. A set of plingo iterative animations created from each frame of step 2

Output

Explanation

As the result of this experiment is not easily understandable, here's some explanation. 

                
                
                
                Image 1
The code itself basically created a matrix of output images. On the x-axis it has the iterations of the second loop. On the y-axis it has the plingo iterations created from each frame of the outer loop. Image 1 to the right shows an overview over this image matrix.

The main animation of the video (big image in the center) zig-zags thru that matrix thus creating a linear animation from the three nested loops. But as this animation does not clearly reveal what happens, I added some linear helpers to the sides. These helpers show linear cuts thru the matrix so you can see different stages of the plingo iterations over the second loo

The images below show the relationship between the image matrix and the different animations.
