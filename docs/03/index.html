#Genuary2021

January 3rd

Prompt

Make something human.

Interpretation

Creating something human using a language such as Plingo seemed not possible to me. So I created a tool-chain that emulates one of the most important human abilities - at least to my understanding - associative thinking.

It starts with an image selected by me. From this point on, the implementation does the following:

  • Run the image in Plingo
  • Use an AI to analyze the resulting image
  • Take the most prominent concept (association) and select the next image using this concept
  • REPEAT
Everything but the selection of the first image is done using APIs. No real-human interaction included.

The AI used is clarifai
