Interference patterns.

Interpretation

The classical visualization of an interference pattern usually uses two waves created by two point sources somewhere in the plane.

This plingo code does the same. It as two "initiators" on its canvas. Starting from these initiators, interference may happen. To see the interference in comparison to only a single source, this page shows all 4 possible versions of this "experiment":

  • On top: No initiators => no change
  • Middle left: Only an initiator to the left
  • Middle right: Only an initiator to the right
  • Bottom: Both initiators set
