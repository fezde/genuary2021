#Genuary2021

January 11th

Prompt

Use something other than a computer as an autonomous process (or use a non-computer random source).

Interpretation

Both input images were created using process that introduces randomness because of their nature. Reflections created by taking the pictures introduce another level of uncontrolled color changes. The combination of both effects is close to real randomness.

While the black-and-white image creates the ususal b/w noise, the colorful image is the first one in this row of experiments that actually created colors during its rendering process.

All output in this project was created by FEZ (Felix Kratzer) using Plingo
