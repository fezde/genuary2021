#Genuary2021

January 12th

Prompt

Use an API.

Interpretation

Pareidolia is the human ability to see known structures (e.g. faces) in more or less abstract images. As AI is set up to emulate human behavior esp. in computer vision, I wanted to see if it also develops its own kind of pareidolia. I took an image - of course showing clouds - and gave it an AI-API for recognition. Then I started to distort the image more and more by running it thru several iterations of plingo. Each iteration I presented to the API again to see what it would detect. Find the results below...

The API used is the AI of clarifai

The work of this prompt was heavily inspired by the work on prompt no. 03. It uses the same API and also researches into the human like behavior of AIs...
