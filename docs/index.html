#Genuary2021

GENUARY is an artificially generated month of time where we build code that makes beautiful things.
It happens during the month of January 2021.
For every 24 hour day within this 744 hour timespan, we have prepared a prompt with instructions for you to execute.
Participating in this month I decided to do all the code in Plingo only. Plingo is an image manipulation programming language that I am currently creating. The main idea behind this language is to merge input and code into one. So I think #genuary2021 is the perfect opportunity to test and explore the possibilities and limits of Plingo.

If you want to learn more about Plingo, check its github repository.
